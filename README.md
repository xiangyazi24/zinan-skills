Zinan's operational skills for AI agent orchestration.

| Skill | Purpose |
|-------|---------|

| `xiang` | Master-worker oversight as Xiang proxy — adjudication substrate |

CC-specific tooling (MCP, Telegram, tmux send-keys) and replace with
Codex-native equivalents (file I/O, git, shell).
